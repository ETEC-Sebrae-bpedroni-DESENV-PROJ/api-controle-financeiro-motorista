API - Controle Financeiro para Motoristas

Catálogo dos domínios da API.

Tipo API Descrição
GET /condutor Retorna uma lista com os condutores.
GET /condutor/{id} Encontra um condutor pelo id.
POST /condutor/create Cria um novo condutor (com os dados no corpo da solicitação).
PUT /condutor/update Edita um condutor (baseado no id do corpo da solicitação).
DELETE /condutor/delete Exclui um condutor (baseado no id do corpo da solicitação).
Exemplo JSON:   { "id": 1, "nome": "teste", "sobrenome": "teste", "dataNascimento": "2000-01-01T00:00:00.000Z", "email": "teste@email", "senha": "teste", "telefone": "(11) 1111-1111", "imagem": "AxD//w==", "enderecoCep": "teste", "enderecoLogradouro": "teste", "enderecoNumero": "teste", "enderecoBairro": "teste", "enderecoCidade": "teste", "enderecoEstado": "teste", "enderecoComplemento": "teste" }
